# **User Journey (사용자 여정 지도)**

---

## **Journey 1: 환경 민감 시민 - 김미정 (37세, 주부)**

### **시나리오: 아이의 건강을 위해 대기질을 확인하고 학교 주변 환경을 모니터링하다**

**컨텍스트**: 
- 초등학생 자녀 2명 (8세, 10세)
- 기흥구 반도체단지 인근 아파트 거주
- 큰아이가 천식이 있어 대기질에 매우 민감
- 평일 아침 7시 등교 준비, 방과 후 3시 하교

---

### **Phase 1: 인지 (Awareness)**

**상황**: 아침 7시, 아이를 학교에 보낼 준비를 하면서 오늘 대기질이 궁금함

| 단계 | 행동 | 접점 | 생각/감정 | 페인 포인트 |
|------|------|------|-----------|------------|
| 1. 정보 탐색 | "오늘 미세먼지 어떻지?"라고 생각하며 스마트폰을 켬 | 스마트폰 홈 화면 | 🤔 "날씨 앱을 봐야 하나, 에어코리아를 봐야 하나?" | 여러 앱을 확인해야 하는 번거로움 |
| 2. 앱 오픈 | 용인 그린워치 앱 아이콘 클릭 | 앱 아이콘 | 😊 "이 앱 하나면 되니까 편해" | - |
| 3. 메인 화면 도착 | 지도와 함께 우리 집 근처 대기질이 한눈에 보임 | 메인 지도 화면 | 😮 "와, 지금 우리 동네가 파란색이네! 좋음이야" | - |

**감정 곡선**: 😐 → 😊 → 😌 (안도)

---

### **Phase 2: 탐색 (Exploration)**

**상황**: 우리 동네는 괜찮은데, 아이 학교 쪽은 어떨까?

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
| 4. 지도 이동 | 두 손가락으로 지도를 확대하며 아이 학교(보라초등학교) 방향으로 이동 | 지도 인터랙션 | 🤔 "학교 근처는 어떨까?" | 즐겨찾기 위치 저장 기능 |
| 5. 측정소 확인 | 학교 근처 측정소 마커를 탭함 | 측정소 마커 | 😊 "여기도 좋음이네, 다행이다" | - |
| 6. 상세 정보 | 팝업에서 PM2.5: 15㎍/㎥ 확인 | 상세 정보 모달 | 😌 "천식 있는 애한테는 20 이하가 좋다고 했는데" | 건강 가이드라인 제공 |
| 7. 히스토리 확인 | "지난주는 어땠지?" 하며 시간 필터를 "지난 7일"로 변경 | 시간 필터 드롭다운 | 🤔 "요즘 계속 좋았구나" | - |
| 8. 차트 확인 | 라인 차트에서 지난주 추이를 확인 | 시계열 차트 | 😊 "꾸준히 좋아지는 추세네" | 예측 정보 제공 |

**감정 곡선**: 🤔 → 😊 → 😌 (만족)

---

### **Phase 3: 행동 (Action) - 알림 설정**

**상황**: 앞으로도 매일 확인하기 번거로우니 알림을 받고 싶음

| 단계 | 행동 | 접점 | 생각/감정 | 페인 포인트 |
|------|------|------|-----------|------------|
| 9. 설정 진입 | 우측 상단 프로필 아이콘 → "알림 설정" 메뉴 클릭 | 네비게이션 메뉴 | 🤔 "매번 확인하기 귀찮은데 알림 받으면 좋겠다" | - |
| 10. 관심 지역 설정 | "대기질 알림" 토글 ON, "알림 기준: 나쁨 이상", "관심 지역: 기흥구 보라동, 구갈동" 선택 | 알림 설정 화면 | 😊 "우리 집이랑 학교 근처 두 곳 다 설정!" | - |
| 11. 저장 | "저장" 버튼 클릭 | 버튼 | 😌 "이제 나쁠 때만 알려주겠지" | - |

**감정 곡선**: 🤔 → 😊 → 😌 (안심)

---

### **Phase 4: 위기 (Crisis) - 3일 후 오후**

**상황**: 오후 2시, 갑자기 푸시 알림이 울림

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
| 12. 알림 수신 | 📱 "보라동 미세먼지 나쁨 (PM2.5: 52㎍/㎥)" | 푸시 알림 | 😰 "아 큰일이다! 아이 하교 시간인데!" | 즉시 대응 가이드 |
| 13. 앱 재오픈 | 알림을 탭하여 앱 실행 | 알림 → 앱 | 😟 "얼마나 나쁜지 확인해야지" | - |
| 14. 지도 확인 | 지도에서 학교 근처가 주황색으로 표시됨 | 메인 지도 화면 | 😨 "생각보다 심각하네" | 실시간 업데이트 |
| 15. 대응 결정 | 남편에게 전화 "여보, 오늘 애 마스크 꼭 쓰게 하고 실내에서 놀게 해" | 외부 (전화) | 😤 "빨리 개선됐으면..." | 개선 요청 제보 유도 |

**감정 곡선**: 😰 → 😟 → 😨 → 😤 (걱정→결의)

---

### **Phase 5: 제보 (Reporting) - 1주일 후**

**상황**: 저녁 산책 중 학교 뒤편 공사장에서 심한 먼지 발견

| 단계 | 행동 | 접점 | 생각/감정 | 페인 포인트 |
|------|------|------|-----------|------------|
| 16. 문제 발견 | "어? 저기 공사장에서 먼지가 엄청 나는데?" | 현장 (오프라인) | 😠 "애들 학교 바로 앞인데!" | - |
| 17. 제보 시작 | 앱을 열고 메인 화면 우측 하단 주황색 "제보하기" 플로팅 버튼 클릭 | 플로팅 버튼 | 😤 "이거 제보해야겠다" | - |
| 18. 사진 촬영 | 카메라가 자동 실행되어 먼지 날리는 공사장 촬영 (3장) | 카메라 UI | 📸 "증거를 남겨야지" | - |
| 19. 위치 확인 | GPS로 자동 태깅된 위치 확인, 지도에서 핀 위치 미세 조정 | 지도 핀 조정 | 😊 "위치가 자동으로 잡히네" | - |
| 20. 유형 선택 | "대기오염" 선택 | 유형 선택 칩 | - | - |
| 21. 설명 작성 | "보라초등학교 뒤편 공사장에서 비산먼지 심각. 방진막도 없음" | 텍스트 입력 | 😠 "자세히 써야 빨리 조치하겠지" | 음성 입력 지원 |
| 22. 제출 | "제보하기" 버튼 클릭 | 제출 버튼 | 😌 "했다! 이제 시에서 확인하겠지" | - |
| 23. 확인 | "제보가 접수되었습니다. 제보번호: YGW-2025100312345" | 완료 화면 | 😊 "제보번호도 주네. 나중에 확인 가능하겠다" | - |

**감정 곡선**: 😠 → 😤 → 😊 → 😌 (분노→행동→안도)

### **Phase 6: 추적 (Tracking) - 2일 후**

**상황**: 제보 후 처리가 어떻게 되고 있는지 궁금함

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
| 24. 알림 수신 | 📱 "제보 상태가 '검토 중'으로 변경되었습니다" | 푸시 알림 | 😊 "오, 확인은 했구나" | - |
| 25. 상세 확인 | 하단 탭 "내 제보" 클릭 → 제보 목록에서 해당 제보 클릭 | 내 제보 탭 | 🤔 "어떻게 처리되고 있을까?" | - |
| 26. 처리 현황 | "환경정책과에서 검토 중입니다. 현장 확인 예정입니다." | 제보 상세 화면 | 😊 "현장 확인까지 하나봐" | 예상 처리 기간 표시 |
| 27. 공감 확인 | "공감 12명" 확인 | 제보 상세 화면 | 😲 "나만 불편한 게 아니었구나" | 공감자 지역 분포 |

**감정 곡선**: 😊 → 🤔 → 😊 → 😲 (궁금→확인→놀람)

---

### **Phase 7: 해결 (Resolution) - 1주일 후**

**상황**: 제보 후 7일째, 처리 완료 알림 수신

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
| 28. 해결 알림 | 📱 "제보하신 문제가 해결되었습니다" | 푸시 알림 | 😮 "벌써?" | - |
| 29. 결과 확인 | 앱 열어서 제보 상세 화면으로 이동 | 제보 상세 화면 | 🤔 "어떻게 해결됐을까?" | - |
| 30. 조치 내용 | "환경정책과 조치 완료: 해당 공사장에 방진막 설치 및 살수차 배치 명령. 사진 첨부됨" | 조치 내용 섹션 | 😊 "우와, 진짜 조치했네!" | - |
| 31. 비포/애프터 | 시에서 첨부한 방진막 설치 후 사진 확인 | 이미지 갤러리 | 😃 "완전 달라졌다!" | - |
| 32. 포인트 적립 | "환경 개선에 기여하셨습니다. 100P 적립!" | 포인트 알림 배너 | 😊 "포인트도 주네? 뭔가 뿌듯하다" | 포인트 사용처 안내 |
| 33. 감사 인사 | 제보 상세 화면 하단 "감사합니다" 댓글 작성 | 댓글 입력창 | 😊 "담당 공무원분께 고마움을 전해야지" | - |

**감정 곡선**: 😮 → 🤔 → 😊 → 😃 → 😊 (놀람→확인→기쁨)

### **Phase 8: 지속 (Retention) - 2주 후**

**상황**: 모은 포인트로 친환경 상점에서 혜택 받기

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
| 34. 포인트 확인 | 하단 탭 "마켓플레이스" → 우측 상단 "내 포인트 250P" | 마켓플레이스 탭 | 🤔 "포인트가 꽤 모였네? 뭐 할 수 있을까?" | - |
| 35. 상점 검색 | 지도에서 우리 집 근처 파트너 상점 탐색 | 지도 뷰 | 🤔 "우리 동네에 어디 있을까?" | 카테고리 필터 |
| 36. 상점 발견 | "어? 동네 제로웨이스트샵이 여기 등록되어 있네!" | 상점 마커 | 😮 "여기 알긴 했는데 비싸서 안 갔었는데" | - |
| 37. 혜택 확인 | 상점 상세 - "200P 사용 시 10% 할인" | 상점 상세 화면 | 😊 "오, 괜찮은데?" | - |
| 38. 방문 결정 | "주말에 애들이랑 가봐야겠다. 환경도 생각하고 할인도 받고!" | 외부 (생각) | 😄 "일석이조!" | 주변 상점 추천 |

**감정 곡선**: 🤔 → 🤔 → 😮 → 😊 → 😄 (탐색→발견→만족)

## **Journey 2: 청년 환경운동가 - 이준호 (28세, NGO 활동가)**

### **시나리오: 처인구 산업단지 악취 문제를 데이터로 입증하여 정책 제안서를 작성하다**

**컨텍스트**:
- 용인환경운동연합 간사
- 처인구 거주, 시민 제보 기반 환경 감시 활동
- 매달 용인시의회에 환경 정책 제안서 제출
- 데이터와 근거를 중시하는 스타일

---

### **Phase 1: 문제 인식**

| 단계 | 행동 | 접점 | 생각/감정 | 목표 |
|------|------|------|-----------|------|
| 1. 민원 접수 | 단체 이메일로 처인구 주민 3명에게서 "남사읍 산단에서 밤마다 악취" 제보 받음 | 이메일 | 🤔 "남사읍에서 또 악취 민원이네" | 패턴 파악 |
| 2. 앱 실행 | 용인 그린워치 앱 실행 | 앱 | 💼 "앱에도 비슷한 제보 있을까?" | 데이터 수집 |
| 3. 지도 탐색 | 지도를 남사읍으로 이동, 제보 마커 필터를 "악취"로 설정 | 지도 + 필터 | 😮 "와, 생각보다 많네" | 시각화 |
| 4. 제보 클러스터 | 남사읍 산업단지 주변에 악취 제보가 밀집되어 있음 (빨간색 클러스터) | 지도 마커 | 😠 "이 정도면 심각한 수준인데" | 근거 확보 |

---

### **Phase 2: 데이터 수집**

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
| 5. 필터 조정 | 시간 범위를 "지난 30일"로 설정, 유형을 "악취"만 선택 | 필터 UI | 💼 "한 달간 얼마나 많이 들어왔나" | - |
| 6. 제보 수 확인 | 지도 상단에 "남사읍 악취 제보 37건" 표시 | 통계 배너 | 😲 "한 달에 37건이나!" | CSV 다운로드 |
| 7. 리스트 전환 | 하단 탭을 "피드"로 전환하여 제보 목록 확인 | 피드 탭 | 🤔 "구체적인 내용을 봐야겠다" | - |
| 8. 제보 읽기 | 각 제보의 사진, 설명, 시간대 확인 | 제보 카드 | 😠 "거의 매일 밤 10시 이후네" | 시간대 분석 차트 |
| 9. 공통점 발견 | "화학약품 냄새", "야간", "바람 방향" 등 키워드가 반복됨 | 제보 설명 | 💡 "패턴이 보이는데, AI 분석이 있나?" | AI 분석 기능 |

---

### **Phase 3: AI 리포트 활용**

| 단계 | 행동 | 접점 | 생각/감정 | 기회 요소 |
|------|------|------|-----------|-----------|
